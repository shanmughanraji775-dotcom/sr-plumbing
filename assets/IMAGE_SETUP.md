### 3. Payment QR Code
- **File Name:** `qr-code.png`

## How to Add Images

1. Save your logo image as `logo.png` in this folder
2. Save your profile picture as `profile.jpg` in this folder
3. Save your payment QR code as `qr-code.png` in this folder

The website will automatically display these images once they are in place.

## File Paths
- Logo: `assets/logo.png`
- Profile: `assets/profile.jpg`
